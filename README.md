# Cards with a Frosted Glass Effect

A project I designed to practice HTML and CSS. Three cards with a glassmorphism effect and a responsive layout for mobile.

## Technologies
HTML5
CSS3 (flexbox, backdrop-filter, media queries)

## What's inside
3 cards in a row on a wide screen
Frosted glass effect (backdrop-filter)
Responsive: on a phone, the cards are stacked one under the other
Smooth hover effects (the card rises, the button expands)

Link: [https://zx-del11.github.io/my-project/]

## Features
No JS - only HTML and CSS
Used style normalization (normalize.css)
Background - an image that is fixed when scrolling
